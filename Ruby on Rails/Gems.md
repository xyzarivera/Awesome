# Installing Gems
1. Open Gemfile
2. paste gemfile code to Gemfile
3. Run `bundle install` to fetch gems
4. Restart rails server
5. Check gem documentation to properly install gem

## Sources
rubygems.org

  - HTML pre-processor
  - change .erb extension to .yaml
2. simple_form
